# 試算表の仕様（添付画像から読み取り）

## 全体構成
- **左側**: 貸借対照表（B/S）科目
- **右側**: 損益計算書（P/L）科目
- 2列構成で表示

## 各列の構成
各列には以下の項目が表示される:
1. **勘定科目名**: 科目名
2. **期首残高**: 前期末時点の残高
3. **借方**: 当期の借方合計
4. **貸方**: 当期の貸方合計
5. **期末残高**: 期首残高 + 借方 - 貸方
6. **構成比**: 期末残高の構成比率（%）

## 貸借対照表（B/S）科目
左側に表示される科目:
- **資産の部**
  - 流動資産
    - 現金
    - 普通預金
    - 定期預金
    - 売掛金
    - 未収入金
    - 貸付金
    - その他流動資産
  - 固定資産
    - 建物
    - 車両運搬具
    - 工具器具備品
    - 土地
    - その他固定資産
  
- **負債の部**
  - 流動負債
    - 買掛金
    - 未払金
    - 未払費用
    - 預り金
    - その他流動負債
  - 固定負債
    - 長期借入金
    - その他固定負債

- **純資産の部**
  - 資本金
  - 繰越利益剰余金

## 損益計算書（P/L）科目
右側に表示される科目:
- **収益の部**
  - 売上高
  - その他収益

- **費用の部**
  - 売上原価
  - 販売費及び一般管理費
    - 給料手当
    - 法定福利費
    - 福利厚生費
    - 旅費交通費
    - 通信費
    - 水道光熱費
    - 消耗品費
    - 地代家賃
    - 減価償却費
    - 支払手数料
    - その他経費

## 実装方針
1. 勘定科目マスタに「科目区分」フィールドを追加（資産、負債、純資産、収益、費用）
2. 期首残高の計算ロジックを実装
3. 試算表のルーティングでB/SとP/Lを分けて集計
4. テンプレートを2列構成に変更
